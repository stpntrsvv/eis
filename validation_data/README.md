# Validation data workspace

Эта папка хранит воспроизводимый контур проверки EIS Solver на открытых экспериментальных данных.

- `manifest.json` — источники, версии, лицензии и назначение корпусов;
- `raw/` — загруженные исходники, исключённые из Git из-за размера;
- `artifacts/` — машинные результаты CLI, также исключённые из Git;
- `reports/` — компактные отчёты и выводы, которые можно хранить в репозитории.

Сырые данные следует получать только по ссылкам из manifest. При публикации результатов необходимо указывать авторов и DOI исходного набора.
