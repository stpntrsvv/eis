# Production-калибровка положительного diffusion-gate

Дата: 2026-07-17.

## Замороженное правило

Пороги после предыдущей репликации не менялись:

```text
bootstrap family winner = inductive_diffusion
family stability >= 90%
ΔBIC(diffusion против лучшей недиффузионной семьи) >= 10
```

Правило является положительным и асимметричным. Оно может поддержать семейство
`inductive_diffusion`, но не выбирает `W/Wo/Ws` и не доказывает отсутствие
диффузионного вклада.

## Новый отрицательный корпус

Сгенерированы 60 независимых спектров истинной схемы
`L0-R0-p(R1,CPE0)`. Между спектрами варьировались:

- все параметры базовой схемы;
- 41, 61 и 81 частотная точка;
- окна `1e-3–1e4`, `1e-2–1e5` и `1e-1–1e4` Hz;
- относительный шум `0,25%`, `1%` и `2%`;
- отсутствие выбросов либо `2%` крупных выбросов.

Для каждого спектра выполнен bootstrap с 10 повторами и теми же четырьмя
кандидатами: base, `W`, `Wo`, `Ws`.

## Результат

- завершено: `60/60`;
- ошибки выполнения: `0`;
- голый family bootstrap winner верен: `58/60`;
- при пороге bootstrap `>=90%` возникло одно уверенное ложноположительное
  diffusion-решение;
- замороженный `bootstrap + ΔBIC` gate: `0/60` ложноположительных рекомендаций;
- односторонняя точная 95%-граница false-positive rate: `4,8703%`.

Таким образом, заранее установленный критерий `<5%` пройден без настройки
порогов по новому корпусу. Одновременно результат показывает практическую роль
BIC-части gate: один bootstrap-порог этот контроль не прошёл.

## Решение

Положительный gate перенесён в reliable inference со следующими ограничениями:

1. Он действует только при явной конкуренции `inductive_diffusion` с другой
   семьёй.
2. Рекомендация возвращается только для `inductive_diffusion`.
3. `recommended_topology` для `W/Wo/Ws` всегда остаётся `null`.
4. Победа базовой схемы возвращает отказ, а не утверждение об отсутствии
   диффузии.
5. В JSON сохраняются использованные пороги, фактический `ΔBIC` и признак
   `positive_only`.

Это калиброванный инженерный gate на замороженной синтетической области, а не
апостериорная вероятность и не физическое доказательство диффузии.
