# Лабораторные шаблоны EC-Lab

Эта папка предназначена для небольших вспомогательных инструментов и
эталонных `.mps`, но не входит в вычислительное ядро EIS Solver.

Текущие файлы `CP_polarize_curves_template.mps` и
`PEIS_eis_template.mps` сохранены EC-Lab 11.43 для BioLogic SP-50e. Сейчас
они содержат одинаковую последовательность связанных техник:

1. PEIS;
2. Chronopotentiometry, `50 µA`, `10 s`.

Различается только внутреннее поле `Filename`. Поэтому первый файл пока нельзя
считать эталоном чистого CP.

Особенности формата, которые должен сохранять будущий генератор:

- однобайтная Windows-кодировка;
- окончания строк CRLF;
- десятичная запятая;
- порядок и ширина текстовых полей;
- служебные параметры переходов и циклов.

До появления эталонов чистого CP и многоступенчатого CP безопасно менять
только значения внутри существующей последовательности `PEIS → CP`. Шаблоны
нельзя автоматически отправлять на прибор без ручной проверки диапазонов,
тока, времени, compliance и защитных ограничений в EC-Lab.
